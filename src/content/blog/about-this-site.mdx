## Site Construction via Components

Components are built in `.astro` files. 
This file type provides an easy way to write code that can get 
exceuted on the server and prerendered before serving the page.

### Component examples in this site


**Skills Card**

import SkillsCard from '../../components/SkillsCard.astro'

<SkillsCard />

**Technical Card**

import TechnicalCard from '../../components/TechnicalCard.astro'

<TechnicalCard />

**Blog Post Row**
```astro
<PostRow 
  title={post.data.title}
  date={post.data.pubDate}
  url={post.slug}
/>
```
